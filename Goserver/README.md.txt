Ica05


Vi har laget en webserver som hjelper med å finne universiteter i Norge ved hjelp av api keys.
Kilden vi bruker som api er googlemaps og openweather. Ideen om nettsiden er at det skal være enkelt å kunne finne frem universiteter i Norge og på samme tid kunne finne ut hvordan været er dit ved hjelp av en annen api. 
Selve dokumentasjonen til kodingen ligger på server.go filen.




Avhengighet
Vi bruker martini og martini render for å kunne kjøre serveren og hente inn templaten.

* github.com/go-martini/martini
* github.com/martini-contrib/render



Ekstra informasjon:

Vi har så langt ikke fått til å bruke openweather apin. Dette er noe som vi skal prøve å fikse på så fort som mulig.
